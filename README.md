# Task 3: CSS Selectors

This project is a part of the MERN Stack course by *TuteDude*. The goal of this task is to demonstrate the practical application of different CSS selectors, including element, class, ID, and descendant selectors, to style HTML elements.

## Features
- *Semantic HTML:* Used various heading tags ($h1$, $h2$, $h3$), paragraphs, and lists.
- *CSS Styling:* - Applied styles using *Class Selectors* (.head).
  - Overridden styles using *ID Selectors* (#bglime, #para).
  - Used *Element Selectors* for general paragraph styling.
  - Used *Descendant Selectors* to style specific words within lists without using classes or IDs.
- *Responsive Layout:* Basic clean structure for browser testing.

## Technologies Used
- HTML5
- CSS3

## How to Run
1. Download or clone this folder.
2. Open the index.html file in any modern web browser (Chrome, Firefox, Edge).
